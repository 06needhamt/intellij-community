Reports conditions like if(Optional.isPresent()) which could be rewritten in functional style. Functional style is shorter and easier to read.

Example:

  if (str.isPresent()) str.get().trim();

After the quick fix is applied the result looks like:

  str.ifPresent(String::trim);

This inspection only reports if the language level of the project or module is 8 or higher.
