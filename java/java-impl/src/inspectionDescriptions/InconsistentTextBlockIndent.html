Reports text blocks with indent that consists of both spaces and tabs. Such cases might be misleading since one space is treated as one tab during text block processing.

For example (here spaces are visualized as · and tabs as ␉, one tab takes 4 spaces in editor):


String colors = """
····················red
␉   ␉   ␉   ␉   ␉   green
····················blue""";

After printing such string the result would be:


red
···············green
blue

This inspection only reports if the configured language level is 15 or higher.

New in 2021.1
