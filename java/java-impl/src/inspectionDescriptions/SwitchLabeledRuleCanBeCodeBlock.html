Reports rules of switch expression or enhanced switch statement which can be converted to a code block.

Example:

  String message = switch (errorCode) {
    case 404 -> "Not found!";
    ...
  };

After the quickfix is applied the result looks like:

  String message = switch (errorCode) {
    case 404 -> {
                yield "Not found!";
            }
    ...
  };

This inspection only reports if the language level of the project or module is 14 or higher

New in 2019.1
