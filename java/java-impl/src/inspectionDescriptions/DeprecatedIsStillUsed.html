Reports deprecated classes, methods and fields which are used in your code nonetheless.
For example: 

class MyCode {
  @Deprecated
  void oldMethod() {}// warning: "Deprecated member is still used"

  void newMethod() {
    oldMethod(); // forgotten usage
  }
}
Usages from within deprecated places are ignored.

NOTE: Due to performance reasons, a non-private member is checked only when it's name rarely occurs in the project.
