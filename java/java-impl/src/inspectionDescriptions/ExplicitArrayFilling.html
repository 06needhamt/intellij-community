Reports loops which could be replaced with the Arrays.setAll() or Arrays.fill() calls. This inspection suggests replacing loops with Arrays.setAll() if the language level of the project or module is 8 or higher. Replacing loops with Arrays.fill() is possible with any language level.

For example:

  for (int i=0; i<array.length; i++) {
     array[i] = calc(i);
  }

Will be replaced with:

  Arrays.setAll(array, this::calc);

New in 2017.1
