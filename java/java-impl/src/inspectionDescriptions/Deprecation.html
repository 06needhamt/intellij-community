Reports used deprecated code, e.g. new Thread().stop();. By default it doesn't report deprecated code when it's impossible or hard to avoid it, e.g. the following code won't be reported: 
  abstract class A {
     @Deprecated
     abstract void m();
  }
  class B extends A {
     @Override
     void m() {
        //doSmth;
     }
  }

Use the checkboxes below to enable this inspection inside deprecated members, overrides of abstract deprecated methods, non-static import statements, methods of deprecated classes or same top level classes.
