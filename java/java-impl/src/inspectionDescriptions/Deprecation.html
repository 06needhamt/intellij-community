Reports usages of deprecated API (classes, fields, and methods), e.g. new Thread().stop();. By default, it doesn't produce a warning if it's impossible or hard to avoid it e.g., the following code won't be reported: 
  abstract class A { //library code
     @Deprecated
     abstract void m();
  }
  class B extends A { //project code
     @Override
     void m() {
        //doSmth;
     }
  }

Use the checkboxes below to enable this inspection inside deprecated members, overrides of abstract deprecated methods, non-static import statements, methods of deprecated classes or same top level classes.
