Reports the code where an unchecked warning is issued by the compiler. Every unchecked warning represents the potential for a ClassCastException at runtime.

Example:

  List items = Arrays.asList("string", "string");
  List<Integer> numbers = Collections.unmodifiableList(items);

The quick-fix tries to generify and put all missed types in the current file:

  List<String> items = Arrays.asList("string", "string");
  List<Integer> numbers = Collections.unmodifiableList(items);
