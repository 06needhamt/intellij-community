Reports enhanced switch statements and expressions, and suggest replacing them with a traditional switch statement.

Example:

  switch (condition) {
    case foo -> throw null;
    default -> b = true;
  }

After the quick-fix is applied the result looks like:

    switch (condition) {
    case foo:
      throw null;
    default ->
      bar = true;
      break;
  }
