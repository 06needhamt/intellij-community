Reports cases where the static Integer.compare() method or similar methods can be used instead of more verbose or less efficient constructs. If x and y are already boxed integers, then x.compareTo(y) is suggested.

Example:

  int z = x > y ? 1 : x < y ? -1 : 0;

After the quick-fix is applied, the result looks like this:

  int z = Integer.compare(x, y);

Double.compare and Float.compare methods appeared in Java 1.4, methods for other primitive types are available since Java 1.7.
