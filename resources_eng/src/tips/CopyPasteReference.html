Edit | Copy Reference action can be used to insert reference to field/method/classs/file into current position in the editor.
For example, go to Collections.java JDK class, position to the EMPTY-MAP field and press &shortcut:CopyReference;:

Then return to your Java source and press &shortcut:$Paste;:

You can also copy references in GotoClass/GotoSymbol/GotoFile dialogs.
Just press &shortcut:$Copy; on any element in the lookup list.
