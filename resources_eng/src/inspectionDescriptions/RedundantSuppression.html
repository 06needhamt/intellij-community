This inspection reports usages of
  • @SuppressWarning annotation, or
  • // noinspection line comment, or
  • /** noinspection */ JavaDoc comment
which can be safely removed because inspection they affect is no more applicable in this context.

For example: 

class C {
 // symbol is no more unused,
 //  but annotation is still around
 @SuppressWarning("UNUSED_SYMBOL")
 private boolean CONST = true;
 void f() {
    CONST = false;
 }
