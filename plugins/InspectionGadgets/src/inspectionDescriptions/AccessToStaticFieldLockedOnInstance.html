Reports on any access to a non-constant static field which is locked on either this or an instance field of this.

Locking a static field on instance data does not prevent the field from being modified by other instances, and thus may result in surprising race conditions.

Example:


  static int num;
  public void foo() {
      synchronized (this) {
          System.out.println(num); // warning
      }
  }
