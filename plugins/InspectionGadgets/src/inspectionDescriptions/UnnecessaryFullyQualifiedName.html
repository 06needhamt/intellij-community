Reports fully qualified class names which can be shortened. The quick-fix shortens fully qualified names and adds import statements if necessary.

Example:

  class ListWrapper {
    java.util.List<Integer> l;
  }

After the quick-fix is applied the result looks like:

  import java.util.List;
  class ListWrapper {
    List<Integer> l;
  }

In Settings | Editor | Code Style | Java | Imports two settings can be found which influence this inspection. Use the settings there to specify if references to inner classes should be qualified with the outer class and if fully qualified names are allowed in javadoc.

Use the checkbox below to ignore fully qualified names inside the Java 9 provides and uses module statements.
