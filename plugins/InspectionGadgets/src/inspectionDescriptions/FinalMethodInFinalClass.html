Reports final methods in final classes.

Since final classes cannot be inherited, marking a method as final may be unnecessary and confusing.

Example:

record Bar(int a, int b) {
  public final sum() { 
     return a + b;
  }
}

After the quick-fix is applied:

record Bar(int a, int b) {
  public sum() { 
     return a + b;
  }
}
As shown in the example, class may be explicitly or implicitly final e.g., enum or record.
