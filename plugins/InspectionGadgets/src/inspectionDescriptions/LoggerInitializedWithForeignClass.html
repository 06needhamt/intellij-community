Reports Logger instances which are initialized with a class literal from a different class than the Logger is contained in. This can easily happen when copy-pasting some contents from another class. 

  public class Paramount {
    protected static final Logger LOG = Logger.getLogger(Critical.class);

    // ... other fields and methods
  }

After the quick fix is applied, the result looks like this:


  public class Paramount {
    protected static final Logger LOG = Logger.getLogger(Paramount.class);

    // ... other fields and methods
  }
