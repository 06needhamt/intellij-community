Reports any fields which may safely be made final. Using final variables makes code more error-prone and might lead to some additional compiler optimisations that will make your program run faster.

Field may safely be made final only if it's private (or defined in local / anonymous class) and:

  • It is static and initialized either in its declaration or in one static class initializer
  • It is non-static and initialized either in its declaration or in one non-static class initializer or in all constructors
