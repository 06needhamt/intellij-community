Reports arrays of enum constants which may be replaced with EnumType.values(). If such enum would be updated later, it would be necessary to update the array as well, though with EnumType.values() no modifications would be required.

Example:


  enum States {
     NOT_RUN, IN_PROGRESS, FINISHED;
  }
  
  handleStates(new States[] {NOT_RUN, IN_PROGRESS, FINISHED});

After the quickfix the code would look like this: handleStates(States.values());

New in 2019.1
