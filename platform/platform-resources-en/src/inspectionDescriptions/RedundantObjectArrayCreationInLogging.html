Reports redundant Object array creation in calls to methods of org.slf4j.Logger so the array creation can be removed using a quick-fix because there are more specific methods. This inspection only handles arrays of the size of 2 or less, bigger arrays are handled with the inspection called Verbose or redundant code constructs | Redundant array creation

Example:
LOGGER.info("{}, {}", new Object[]{"Hello", "World"})
is converted to
LOGGER.info("{}, {}", "Hello", "World")

Since 2020.2
