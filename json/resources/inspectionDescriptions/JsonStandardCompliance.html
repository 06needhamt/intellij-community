Checks a JSON file against the language specification in the following aspects:
  • No line and block commentaries (configurable).
  • No multiple top-level values (expect for JSON Lines files, configurable for others).
  • No trailing commas in objects and arrays (configurable).
  • No single quoted strings.
  • Property keys are double quoted strings.
  • No NaN and Infinity/-Infinity numeric values as floating point literals (configurable).
